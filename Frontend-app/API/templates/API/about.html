About

FireGuard is a web application that allows people to check the fire risk for their homes at any given time, based on longitude and latitude.
Users can view the location of their homes on a map and receive notifications when time to flashover is below a certain threshhold near the specified location.
With FireGuard, users can stay informed and take action to protect themselves from the threat of fires.

The FireGuard project is still in beta. This means we're only providing risk calculations for the a small amount of locations. Check out the map page to see which locations we currently support.

How does it work?

FireGuard uses a combination of data sources to calculate the fire risk for a given location.
The main data sources are weather data and forecasts from the MET and Storm. The data is fetched once every 'x' hours, and is used to calculate the current fire risk for a given location via the time to flashover at that location.
Time to flashover is the time it takes from conflagration to a full-scale fire at a location.

FireRisk Calculations

The calculations are done with an implementation by R.D. Strand and L.M. Kristensen, the implementation is described in the submitted paper:
An implementation, evaluation and validation of a dynamic fire and conflagration risk indicator for wooden homes.
