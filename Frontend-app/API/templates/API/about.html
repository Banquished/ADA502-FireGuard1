About FireGuard

FireGuard is a school project in the course ADA502.
The goal of the project is to create a web application that can track and monitor the risk of fires setting out in wooden-buildings in Norway.

It is built using the following technologies:

  • Django
  • React
  • Docker
  • SQLite

It is currently hosted locally with SQLite databases.

The project follows a microservice architecture and is a part of Lars Michael Kristensens FRCM project.
