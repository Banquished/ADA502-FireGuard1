FireGuard

FireGuard Maps Integration

FireGuard is a web application that allows users to track and monitor the fire risk for their homes. Users can view the location of their homes on a map and receive notifications when time to flashover is below a certain threshhold near the specified location. With FireGuard, users can stay informed and take action to protect themselves from the threat of fires.

FireGuard Locations

The FireGuard project is still in beta. This means we're only providing risk calculations for the following locations

  • Oslo
  • Bergen
  • Stavanger
  • Kristiansand

We're working hard to expand our coverage to more locations. If you want to suggest important locations, please contact us.
